# CovoitAir AI Image Prompt Pack

## App Analysis

This app is not a generic taxi product. It is a mobile-first airport carpooling platform called `CovoitAir` with two-sided flows:

- passengers search or request rides to and from airports
- drivers publish rides, manage seats, luggage, and bookings
- both sides use chat, notifications, ratings, and profiles
- the product emphasizes trust, verified identity, route clarity, and cost savings
- the UI style is clean, rounded, modern, bright, and friendly

## Visual DNA To Keep Consistent

- primary color: vivid travel blue `#3B82F6`, deep blue `#1E3A8A`, accent blue `#007AFF`
- secondary accents: green for success, soft purple for request/matching moments, amber for trust/warnings
- atmosphere: optimistic, safe, premium, human, efficient
- geography feel: Europe / North Africa airport mobility
- composition: mobile-app-friendly, lots of negative space, rounded shapes, soft gradients, subtle map or route motifs
- people: diverse travelers, drivers, young professionals, students, families, light luggage and suitcases
- avoid: taxi-only look, luxury chauffeur look, cyberpunk, dark gritty scenes, cluttered backgrounds, generic stock-photo feeling

## Base Style Prompt

Use this at the end of every prompt if your image model accepts long prompts:

`modern travel-tech illustration, premium mobile app branding, rounded shapes, soft gradient lighting, blue and white palette with subtle green and purple accents, clean composition, high detail, friendly human-centered design, polished startup aesthetic, suitable for a React Native mobile app, no text, no watermark, no logo unless requested`

## Negative Prompt

Use this if your tool supports negative prompts:

`low quality, blurry, distorted faces, extra fingers, messy background, crowded composition, dark horror mood, cyberpunk, heavy shadows, aggressive red palette, taxi sign, limousine, watermark, UI text, random letters, duplicated luggage, malformed cars, bad anatomy`
